# 个人日记系统

## 前言

个人的毕业设计作品，Spring boot + Vue结构

### 目录说明

Images README图片资源  
Web vue前端项目  
src springboot后端代码  
static 存储于后端的资源文件  
lifediary.sql 为数据库文件

## 项目功能

- 账号系统
  - 注册页
  - 登录页
- 日记系统
  - 日记列表
  - 日记编写
  - 时光轴
